# Purpose — {{WIKI_NAME}}

## What this wiki is for

Capture knowledge from consulting engagements: client context, recommendations made, lessons learned, patterns that repeat across clients. The wiki is the long-term memory across engagements, not the deliverable for any one client.

Decisions this wiki supports: which patterns to apply to a new client, which approaches to avoid, which recommendations have aged well, which haven't.
