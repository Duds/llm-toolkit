# OM Canvas Slides

## Purpose

Build a single-file HTML slide deck for Operating Model Canvas presentations.

## Quick reference

- Output: single `.html` file, no dependencies, opens directly in browser
- Slide types: Title | POLISM Canvas | Section | Detail | Summary
- Brand: chains to `dcceew-brand-guide` or `dale-style-guide` depending on audience
- POLISM cells: Purpose, Organisation, Location, Information, Services, Management
- Navigation: keyboard arrows, click, or touch swipe
- Print: CSS print media query produces A4 landscape output
