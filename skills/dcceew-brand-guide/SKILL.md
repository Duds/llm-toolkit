# DCCEEW Brand Guide

## Brand tokens

```css
--brand-primary:      #00698C;   /* DCCEEW teal */
--brand-secondary:    #004B6B;   /* dark teal */
--brand-accent:       #6ABD45;   /* green */
--brand-neutral:      #F0F0F0;   /* light grey */
--brand-text:         #333333;   /* body */
--brand-white:        #FFFFFF;
--brand-font-sans:    'Public Sans', Arial, sans-serif;
--brand-font-heading: 'Public Sans', Arial, sans-serif;
```

## Typography

- Headings: Public Sans Bold, sentence case
- Body: Public Sans Regular, 11–12pt
- Never use ALL CAPS for headings
- Minimum font size: 9pt for footnotes

## Colours

| Use | Colour | Hex |
|-----|--------|-----|
| Primary headings, key elements | DCCEEW Teal | #00698C |
| Dark backgrounds, contrast | Dark Teal | #004B6B |
| Accents, highlights | DCCEEW Green | #6ABD45 |
| Tables, backgrounds | Light Grey | #F0F0F0 |
| Body text | Charcoal | #333333 |

## Layout principles

- A4 or A4 landscape for documents
- 20mm margins minimum
- DCCEEW logo top-left on all materials
- Footer: document title | date | classification | page number
- Classification: OFFICIAL (default) — mark all materials

## Document standards

- All documents must carry a classification marking
- Use sentence case for headings throughout
- Tables: teal header row (#00698C, white text), alternating grey rows
- Images: use Commonwealth-licensed or DCCEEW-owned images only
