# Provenance Registry

Source material tracking for `raw/` folder.

## Format

| source_path | raw_filename | ingest_date | ingest_context | processed_to |
|-------------|--------------|-------------|----------------|--------------|
| Original location | Name in raw/ | YYYY-MM-DD | Why this matters | knowledge/ path |

## Entries

| source_path | raw_filename | ingest_date | ingest_context | processed_to |
|-------------|--------------|-------------|----------------|--------------|

## Notes

- `source_path`: Relative path from wiki root or absolute path
- `raw_filename`: Filename as stored in `raw/`
- `ingest_date`: When copied to `raw/`
- `ingest_context`: Brief note on significance
- `processed_to`: Path in `knowledge/` if processed, or `pending`
