# DCCEEW Costing

## Purpose

Generate a DCCEEW ICT Costing Worksheet (Template v32 format) as an Excel workbook.

## Read the full skill

This stub routes to the full implementation.
Read `/mnt/skills/user/dcceew-costing/SKILL.md` for complete instructions,
including sheet structure, formula patterns, and rate card values.

## Quick reference

- Output: `.xlsx` file matching Template v32 structure
- Phases: PLAN / BUILD / IMPLEMENT / CLOSE / ONGOING
- Sheet 1: Overview and summary
- Sheet 2: Intangible Asset Guidance
- Sheet 3: Data Entry (effort days by role by phase)
- Rate card: current DCCEEW labour rates by APS/SES level
- Always include FY spend profile across the program lifecycle
