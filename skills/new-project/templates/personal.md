## Voice and style

All writing in this project follows Dale's personal style. Use the `dale-style-guide`
skill before producing any content that will be published or shared.

Key principles:
- Direct, no filler — lead with the point
- Concrete over abstract — show, don't describe
- Conversational but precise — no corporate language
- Short paragraphs, active voice
